# MagicJinn's Minecraft Tweaks

Datapack with small crafting and loot tweaks.

## Recipes

- **Chains** — Chain output amount 1 > 8
- **Slime** — Slime block slime ball requirement 9 > 4
- **Recycling smelting** — Compass smelting result iron ingot; Lodestone smelting result compass.
- **Ore block smelting** — Raw iron/gold/copper blocks smelt or blast into metal blocks.
- **Missing blasting** — Blast recipes for stone, smooth stone, nether brick, deepslate, glass, brick.
- **Firework duration** — Craft fireworks with chosen flight duration 1-3 > 1-8.

## Loot

- **Ghast** — Added Fire Charge to ghast loot table.
- **Emerald ore** — Increased emerald drop amount from 1 to 8.
